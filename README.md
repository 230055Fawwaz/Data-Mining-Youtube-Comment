# Data-Mining-Youtube-Comment
Mini Proyek Data Mining B 2025

User Guide:
1. Jalankan file youtube_scraper.py terlebih dahulu untuk mengambil komentar youtube.
2. Jalankan file preprocessing.py untuk membersihkan data komentar.
3. Jalankan file app.py untuk visualisasi hasil dengan mengetikkan 
   " streamlit run app.py " di terminal.
